# Billin frontend

## Requirements

``` 
node@12.16.0
npm@6.13.6
```

## Install dependencies

``` bash
npm install
```

## Run tests

``` bash
npm test
```

### Run service

``` bash
npm start
```
After this, the application is running in http://localhost:8080 need launch the backend project https://github.com/PhyrionX/nodetest
